#Hexo博客搭建

一，首先在github建立仓库

二，执行以下步骤

step 1.brew install node

step 2.npm install -g hexo

step 3.hexo -v
如果安装成功可以执行以上命令，查看版本

step 4.本地新建博客目录 并在此目录下执行 hexo init

step 5.hexo generate
生成静态页面

step 6.hexo server
本地启动
localhost:4000访问本地页面

三，theme修改与github布署


修改_config.yml theme字段可以修改博客主题。默认主题为landscape

当然，前提你需要下载主题，可以将主题clone下来。

1.iissnan/hexo-theme-next
2.litten/hexo-theme-yilia
3.TryGhost/Casper
4.wuchong/jacman
5.A-limon/pacman
6.daleanthony/uno
7.orderedlist/modernist
8.AlxMedia/hueman
9.kathyqian/crisp-ghost-theme
10.xiangming/landscape-plus

修改_config.yml
deploy:
type: git
repository: https://...git
branch: master

执行：hexo deploy 完成布署
